## Run Locally 

Clone the project
```plaintext
git clone https://github.com/JaviNGD/shopping-list-react.git
```

Go to the project directory
```plaintext
cd shopping-list-react
```

Install dependencies
```plaintext
  npm install
```

Start the server
```plaintext
  npm start
```
